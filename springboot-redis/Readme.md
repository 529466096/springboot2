## Redisson 分布式锁实例

详见 [Redisson实现分布式锁](E:\NoodlesGuide\6-系统设计\1-Redisson实现分布式锁.md)

 使用分布式锁需要考虑哪些问题

- 互斥性：在任意时刻，只能有一个进程持有锁
- 防死锁：即使有一个进程在持有锁的期间崩溃而未能主动释放锁，要有其他方式去释放锁从而保证其他进程能获取到锁。
- 加锁和解锁的必须是同一个进程
- 锁的续期问题。
